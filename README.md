# Сервис для рассылки и подписания документов

Данный проект представляет собой backend-сервис для рассылки и подписания документов внутри организаций с использованием ролевой модели и подтверждения подписи пользователя.

## Описание проекта

Система позволяет:
- регистрировать пользователей;
- создавать организации (предприятия);
- создавать отделы внутри организаций;
- приглашать сотрудников в организации;
- назначать роли сотрудникам в отделах;
- отправлять документы на подписание другим сотрудникам;
- подписывать документы с использованием подписи, сформированной на основе номера телефона пользователя;
- подтверждать подписание документа через выбранный канал уведомлений (email / SMS / мессенджер).

Проект реализован в виде REST API с использованием FastAPI и PostgreSQL.

## Основной функционал
- Регистрация и аутентификация пользователей
- Управление организациями и отделами
- Ролевая модель сотрудников
- Отправка и подписание документов
- Подтверждение подписания документов
- Документация API через Swagger UI

## Используемые технологии
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT
- Swagger (OpenAPI)

## Запуск проекта

1. Установить зависимости:
```bash
pip install -r requirements.txt
```

2. Настроить подключение к PostgreSQL в .venv

3. Запустить сервер:
```bash
uvicorn main:app --reload
```
Swagger UI будет доступен по адресу:
```bash
http://localhost:8000/docs
```
